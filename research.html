Real-Time DDS Communication for Autonomous Vehicles

UNC Computational Robotics Lab

This research focuses on enabling real-time communication between sensors and actuators in autonomous vehicle systems using ROS 2 and Data Distribution Service (DDS). I'm developing and testing solutions using Autoware and AWSIM with Unity to simulate autonomous driving scenarios.

A key contribution is Auto-Reach, a tool I created to facilitate the connection between two open source autonomous vehicle simulation stacks, enabling researchers to more easily test and validate their real-time communication systems.

Formally Verified Python-to-Verilog Compiler

UNC Brain Computer Interface Lab

I'm developing a formally verified compiler that translates Python code into Verilog, enabling seamless hardware-software co-design. This work explores Processing-in-Memory (PIM) hardware architectures specifically optimized for Fast Fourier Transformations (FFT).

The goal is to create a verified toolchain that allows researchers to write high-level Python code and automatically generate correct-by-construction hardware implementations for signal processing applications in brain-computer interfaces.
